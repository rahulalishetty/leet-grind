# 214. Shortest Palindrome

## Problem Statement

You are given a string `s`.

You can convert `s` to a palindrome by **adding characters in front of it**.

Return the **shortest palindrome** you can obtain by performing this transformation.

---

## Example 1

**Input**

```text
s = "aacecaaa"
```

**Output**

```text
"aaacecaaa"
```

---

## Example 2

**Input**

```text
s = "abcd"
```

**Output**

```text
"dcbabcd"
```

---

## Constraints

- `0 <= s.length <= 5 * 10^4`
- `s` consists only of lowercase English letters
