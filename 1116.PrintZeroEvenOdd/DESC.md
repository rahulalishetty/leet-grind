# 1116. Print Zero Even Odd

You are tasked with implementing a class `ZeroEvenOdd` that outputs a sequence of numbers in the format "0102030405..." up to a length of `2n`. The sequence is generated by three threads working together:

- **Thread A**: Calls `zero()` to output zeros.
- **Thread B**: Calls `even()` to output even numbers.
- **Thread C**: Calls `odd()` to output odd numbers.

### Implementation Details

Implement the `ZeroEvenOdd` class:

- **Constructor**: `ZeroEvenOdd(int n)`
  Initializes the object with the number `n`, which determines the length of the sequence.

- **Methods**:
  - `void zero(printNumber)`
    Outputs a zero by calling `printNumber`.
  - `void even(printNumber)`
    Outputs an even number by calling `printNumber`.
  - `void odd(printNumber)`
    Outputs an odd number by calling `printNumber`.

### Example

#### Input:

```
n = 2
```

#### Output:

```
"0102"
```

#### Explanation:

Three threads are executed asynchronously:

- One thread calls `zero()`.
- Another thread calls `even()`.
- The third thread calls `odd()`.

The correct output sequence is `"0102"`.

#### Input:

```
n = 5
```

#### Output:

```
"0102030405"
```

### Constraints

- `1 <= n <= 1000`
- The sequence must be generated by coordinating the three threads.

### Notes

The `printNumber` function is provided and is used to output numbers to the console.
